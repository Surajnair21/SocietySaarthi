📌 Project Description

SocietySaarthi is a centralized web platform designed to simplify and digitize residential society management for apartments and villa communities. It integrates complaint handling, community engagement, marketplace features, event management, and security systems into a single unified solution.

The platform leverages AI-powered features to enhance complaint submissions, automatically categorize issues, and detect duplicate complaints—reducing redundancy and improving efficiency.

🚀 Key Features
🛠 Complaint Management
Report and track complaints in real-time
AI-enhanced structured complaint generation
Status lifecycle: Pending → In Progress → Resolved
Duplicate complaint detection
🛒 Society Marketplace
Buy, sell, or exchange items within the society
Category-based listings with price options (free/paid)
📢 Events & Announcements
Receive real-time updates and notices
Automated reminders for upcoming events
🚪 Visitor & Delivery Tracking
Log visitor entries with details
Track incoming deliveries efficiently
🔐 Role-Based Access
Separate dashboards for residents, admins, and security staff
Controlled access for sensitive operations
🎯 Objective

To build a smart, scalable, and efficient digital ecosystem that enhances everyday residential society operations using modern web technologies and AI.
